ФУНКЦИИ ПРЕОБРАЗОВАНИЯ ДАННЫХ (python3.8)

1.scalestretch another - функция python
растягивает входящий список 
элементов типа int или float 
в самых контрастных участках 
до указанного размера списка.
аргументы:
(1.список int/float на вход,
2.число элементов в выходном списке int)

2.minimaser list - функция python 
сжимает список элементов типа 
int или float с минимальными 
потреями пиковых значений, 
значений краев и общего среднего 
значения.
аргументы:
(1.список int/float на вход,
2.число элементов в выходном 
списке (8,7,6,5,4,3,2,1))

3.freerange - функция python
принимает два int/float для 
создания краев выходного списка. 
принимает int для деления на него
разницы краев. пример:
первый край: 6
второй край: 78
делим на 11
итог:
[6.0, 12.55, 19.09, 25.64, 
32.18, 38.73, 45.27, 51.82, 
58.36, 64.91, 71.45, 78.0]


ФУНКЦИИ АНАЛИЗА ДАННЫХ (python3.8)

1.navigation one list - функция python
функция предназначена для простого
итерированного анализа связей между
элементами списка int/float.
Описание теста:
каждую новую итерацию пользователь
должен ввести некое число.
в ответ он получит простейшую статистику
всей истории вводов:
  *наибольшее число
  *наименьшее
  *среднее
А так же простейший анализ и прогноз:
  *на какое и уже имеющихся чисел 
   в списке похоже то, что было 
   введено последним
  *какую последовательность чисел 
   стоит ожидать далее (исходя из того,
   какие числа шли после похожего)
Кроме этого на вывод проецируется весь 
список чисел и словарь с введённым числами 
и их последними упоминаниями.
